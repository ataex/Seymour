Seymour

An instance of the seymour client, adding the rendering canvas to the supplied html element.

new Seymour(myElement: Object, options: Object)
Parameters
myElement (Object) The html element.
options (Object) Options for Seymour instance.
Properties
isResizable (boolean) : If true, client canvas will resize with window.
doRequestRender (boolean) : If true, requests will be sent for server renderings.
interactionMode (Object) : Current mode as INTERACTION_ENUM.
serverTexture (boolean) : Request render from server with/without texture.
rotationMatrix (Object) : THREE.Matrix4 storing the rotation of this.models.
container (Object) : Reference to div element containing the client canvas and imgOverlay.
imgOverlay (Object) : Reference to img element holding rendered server frame.
scene (Object) : THREE.Scene .
camera (Object) : THREE.Camera , is THREE.PerspectiveCamera by default.
models (Object) : THREE.Object3D containing models for interaction.
pointLight0 (Object) : THREE.PointLight
pointLight1 (Object) : THREE.PointLight
pointLight2 (Object) : THREE.PointLight
